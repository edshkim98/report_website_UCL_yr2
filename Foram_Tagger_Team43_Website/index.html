Introduction

Foraminifera is microscopic fossils which are about 0.5mm sized. Micropalaeontologists research these by analysing and collecting forams from various depths below the sea-bottom, to forensically reconstruct millions of years of planet-scale events. They are collected and photographed under a microscope, but they also need to be:
-digitally segmented from each other
-sotred in a DB with meta-info
-tagged with morphological information -retrieved from the DB by Micropalaeontologists.

Push the button:

Our main goal is to create an application of automated Foraminifera-Tagger by implementing neural network.


Must Have

-The ability to pick out individual forams when given an image of a group of forams
-The ability to identify the modern species of forams when give an image
-Allow users to make corrections when forams are incorrectly identified
Store the pictures of forams in a database
Allow the users to search for forams based on certain criteria e.g. species

Should Have

-The ability for the machine learning model to learn when a correction by the user has been made
-The ability to identify all species of forams when give an image

Could Have

-Different views for the gallery

Research

There are many other frameworks available such as Django, Flask, Codelgniter and Laravel.

We are also going to use pytorch for building machine learning algorithms as there are
